# DocumentChecksummer

## Summary

Supported implementations for Document Checksummer.

## Notes

Generates a checksum from a document's content to detect changes between crawl sessions. When a document's checksum matches its previous value, the document is considered unchanged and may be skipped. Select one of the supported implementations to configure document change detection.
