# ReferenceDelayResolver

## Summary

Apply different delays per different URL patterns.

## Notes

TODO: Add notes.

## Properties

### defaultDelay

The default delay in milliseconds.

### delayReferencePatterns

Use different delays by corresponding URLs.

### ignoreRobotsCrawlDelay

By default the crawl delay specified in `robots.txt` takes precedence over custom settings. Check to disable robot delay instructions.

### scope

The delay scope.

## Examples

The following example will increase the delay to 10 seconds when  encountering PDFs from a default of 3 seconds.



<ConfigExample
  json={{
    "class": "ReferenceDelayResolver",
    "defaultDelay": 3000,
    "delayReferencePatterns": [
      {
        "delay": 10000,
        "pattern": ".*\\.pdf"
      }
    ]
  }}
  xml={`
<class>ReferenceDelayResolver</class>
<defaultDelay>3000</defaultDelay>
<delayReferencePatterns>
  <delayReferencePattern>
    <delay>10000</delay>
    <pattern>.*\.pdf</pattern>
  </delayReferencePattern>
</delayReferencePatterns>`}
/>
