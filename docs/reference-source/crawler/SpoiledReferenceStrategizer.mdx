# SpoiledReferenceStrategizer

## Summary

Supported implementations for Spoiled Reference Strategizer.

## Notes

Determines the strategy to apply when a previously crawled document can no longer be retrieved (a "spoiled" reference). Strategies include deleting the document from the index, gracing it once before deletion, or ignoring the failure. Select one of the supported implementations to configure this behavior.
