## Properties

### cssSelector

CSS-like selector of an element to take a screenshot of.

### imageFormat

Saved image format supported by the selected WebDriver (e.g., PNG, JPEG, GIF).

### targetDir

Directory location where to save screenshot images. Only applicable when directory is one of the specified targets.

### targetDirField

Name of the metadata field where to store the path to the saved screenshot file.

### targetDirStructure

Directory structure to adopt when saving screenshot files.

### targetMetaField

Name of the metadata field where to store the screenshot image itself in BASE64 format. Only applicable if metadata is one of the selected targets.

### targets

Screenshot file saving destination(s).
