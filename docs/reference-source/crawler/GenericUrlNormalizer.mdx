# GenericUrlNormalizer

## Summary

Generic implementation that should satisfy most URL normalization needs.

## Notes

TODO: Add notes.

## Properties

### normalizations

URL normalization rules. Rules prefixed with an asterisk (*) preserve semantic equivalence, as per the [RFC 3986](http://tools.ietf.org/html/rfc3986) standard.

### replacements

Replace one or more matching portions of URLs with different text.
