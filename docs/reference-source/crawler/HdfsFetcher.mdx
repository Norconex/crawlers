# HdfsFetcher

## Summary

Apache Hadoop File System (HDFS) fetcher.

## Notes

Fetches documents from Apache Hadoop File System (HDFS). Supports credential configuration and loading of additional HDFS configuration resources by name, path, or URL to override default settings.

## Properties

### configNames

Names of configuration resources to be loaded after the defaults, to override any specific HDFS settings.

### configPaths

Paths of local file system configuration files to be loaded after the defaults, to override any specific HDFS settings.

### configUrls

URLs of configuration files to be loaded after the defaults, to override any specific HDFS settings.

### credentials

Username and password to a protected HDFS.

### domain

The domain name for authentication when accessing the remote resource.

### referenceFilters

Filters applied to document references (URLs or file paths) to control which ones are fetched. Documents whose references do not pass the filters are skipped.
