# DelaySchedule

## Summary

Dictates what delay to use at the specified time ranges. Range start value can be greater than the end value.

## Notes

Defines a time-based schedule entry that specifies the crawl delay to apply within certain time ranges. Supports day-of-week, day-of-month, and time-of-day ranges. The start value can be greater than the end value to allow overnight ranges.

## Properties

### dayOfMonthRange

Range of month days. Leave blank for any month day.

### dayOfWeekRange

Range of week days. Leave blank for any week day.

### delay

The delay for this schedule entry, in milliseconds.

### timeRange

Range of day times. Leave blank for any time.
