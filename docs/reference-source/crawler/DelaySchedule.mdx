# DelaySchedule

## Summary

Dictates what delay to use at the specified time ranges. Range start value can be greater than the end value.

## Properties

### dayOfMonthRange

Range of month days. Leave blank for any month day.

### dayOfWeekRange

Range of week days. Leave blank for any week day.

### delay

The delay for this schedule entry, in milliseconds.

### timeRange

Range of day times. Leave blank for any time.
