## Examples

The following example adds URLs from `<script>` tags to the  default list of URLs to be extracted.

<ConfigExample
  json={{
    "class": "HtmlLinkExtractor",
    "tagAttribs": {
      "a": [
        "href"
      ],
      "frame": [
        "src"
      ],
      "iframe": [
        "src"
      ],
      "img": [
        "src"
      ],
      "meta": [
        "http-equiv"
      ],
      "script": [
        "src"
      ]
    }
  }}
/>
