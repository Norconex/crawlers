# FsQueue

## Summary

A local file system queue for storing committer requests as zip files.

## Notes

A local file system queue that stores committer requests as zip files before they are sent to the target committer. Supports configurable batch sizes, per-folder file limits, leftover commit handling on startup, and behavior on commit failure.
