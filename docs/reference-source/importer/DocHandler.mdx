# DocHandler

## Summary

Supported implementations for Doc Handler.

## Notes

A document handler is an importer processing step that acts on a document at a defined stage (pre-parse or post-parse). Select one of the supported implementations — such as transformers, parsers, splitters, or flow control wrappers — to configure how documents are processed.
