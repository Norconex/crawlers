# OcrConfig

## Summary

Configuration for Tesseract OCR text extraction from images and image-containing documents.

## Notes

Configuration for Optical Character Recognition (OCR) using Tesseract. Enables text extraction from images and image-containing documents (such as scanned PDFs). Supports configuring the Tesseract path, language data, image preprocessing, content-type restrictions, file size limits, and various Tesseract-specific options.
