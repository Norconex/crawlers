# ImageTransformer

## Summary

Transforms an image using common image operations.

## Notes

This class should only be used as a pre-parsing handler, on image files. It may also be appropriate to disable parsing of those images if you want to keep the transformed version intact. This can be done with [GenericDocumentParserFactory]{TODO link}.

### Content-types
By default, this filter is restricted to (applies only to) documents matching the restrictions returned by [CommonRestrictions.imageIOStandardContentTypes(String)](TODO link). You can specify your own content types if you know they represent a supported image.

### Image dimension format
For a list of supported image dimension formats, refer to [DimensionConverter](TODO link).
