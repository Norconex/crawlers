# DirStructure

## Summary

Defines the directory structure convention used for on-disk storage.

## Notes

Controls how files are organized into subdirectories on disk. Different structures (e.g., date-based, hash-based) affect how documents and their metadata are laid out in the file system for storage and retrieval.
