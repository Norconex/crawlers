# StorageDiskStructure

## Summary

Supported on-disk directory structure layouts for the storage engine.

## Notes

Controls how the storage engine organizes data on disk. Different layouts (e.g., flat, date-based, or hash-partitioned) affect file distribution, lookup performance, and how stored data is navigated or archived.
