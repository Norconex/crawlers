# SolrClientType

## Summary

Supported Solr client connection types.

## Notes

Specifies how the Solr committer connects to a Solr instance. Available types include standard HTTP client connections and SolrCloud-aware connections that support ZooKeeper-based cluster discovery and load balancing across Solr nodes.
