# DelayRangeDOW

## Summary

A day-of-week range with start and end values for delay scheduling.

## Notes

Defines a range of days of the week (e.g., Monday through Friday) used to apply crawl delay rules during specific days. The `start` and `end` values specify the first and last day of the range, inclusive.

## Properties

### end

The end boundary of the range.

### start

The start boundary of the range.
