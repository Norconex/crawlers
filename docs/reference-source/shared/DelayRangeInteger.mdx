# DelayRangeInteger

## Summary

An integer-based range with start and end values for delay scheduling.

## Notes

Defines a numeric range (in milliseconds) used to set crawl delay boundaries. The `start` value represents the minimum delay and `end` represents the maximum delay, allowing the crawler to apply a randomized or fixed wait within that range.

## Properties

### end

The end boundary of the range.

### start

The start boundary of the range.
