## Examples

### basic

The most basic form, performing an exact match on entire text.  It will match "Paul", but not "paul", and not "I am Paul".

<ConfigBlock
  json={{
    "pattern": "Paul"
  }}
/>

### wildcard

The following will successfully match this sentence:  "It seems Paul and Marc are friends".

<ConfigBlock
  json={{
    "method": "WILDCARD",
    "ignoreCase": true,
    "partial": true,
    "pattern": "paul*mar?"
  }}
/>
